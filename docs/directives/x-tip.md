# x-tip

Creates a simple tooltip.

```html
<span x-tip="Tooltip content">Info</span>
```

## Attributes

Use `left`, `top`, `right` or `bottom` to set the `:position`. Default: `right`.

```html
<span x-tip:position="Tooltip content">Info</span>
```

Add `.onlyClick` to active toggle only by click or touch events.

```html
<span x-tip.onlyClick="Tooltip content">Info</span>
```
